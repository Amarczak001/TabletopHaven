# TabletopHaven

**Instalation**<br />
To install project:
```
npm install.js x
```
where x = amount of games we want to download to database<br />

To install project with test users:
```
npm install-with-users.js x
```
where x = amount of games we want to download to database<br /><br />

**Running project**<br />
To run project:
```
npm run.js
```
